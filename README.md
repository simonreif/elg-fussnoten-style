Da die E-Learning Group einen sehr idiosynkratischen Zitierstil hat und dieser nur als Citavi-Stil zur Verfügung steht, ist hier der Stil als CSL für Menschen, die Zotero verwenden.

Since the E-Learning Group uses a very idiosyncratic citation style that is only available as a Citavi style, here is the style as a CSL file for people who use Zotero.”
